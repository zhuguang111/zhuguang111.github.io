# 数据库配置指南

本文档说明如何在微信云开发控制台中创建所需的数据库集合。

## 创建数据库集合

### 1. 登录云开发控制台

1. 打开微信开发者工具
2. 点击工具栏上的"云开发"按钮
3. 进入云开发控制台

### 2. 创建 drone_detection 集合

1. 点击左侧"数据库"
2. 点击"+"号创建集合
3. 集合名称：`drone_detection`
4. 权限设置：选择"所有用户可读，仅创建者可写"
5. 点击"确定"

### 3. 创建 detection_locations 集合

1. 再次点击"+"号创建集合
2. 集合名称：`detection_locations`
3. 权限设置：选择"所有用户可读，仅创建者可写"
4. 点击"确定"

### 4. 创建 alerts 集合

1. 再次点击"+"号创建集合
2. 集合名称：`alerts`
3. 权限设置：选择"所有用户可读，仅创建者可写"
4. 点击"确定"

### 5. 创建索引

为提高查询效率，需要为集合创建索引：

#### drone_detection 集合索引
1. 点击进入 `drone_detection` 集合
2. 点击"索引管理"
3. 点击"添加索引"
4. 添加以下索引：
   - 索引字段：`location_id`，排序：降序
   - 索引字段：`detected_at`，排序：降序
   - 索引字段：`is_drone`，排序：降序

#### alerts 集合索引
1. 点击进入 `alerts` 集合
2. 点击"索引管理"
3. 点击"添加索引"
4. 添加以下索引：
   - 索引字段：`is_read`，排序：降序
   - 索引字段：`created_at`，排序：降序
   - 索引字段：`is_handled`，排序：降序

## 验证创建结果

创建完成后，应该能看到以下集合：
- drone_detection
- detection_locations
- alerts
