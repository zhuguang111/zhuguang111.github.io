# 数据库配置指南

本文档说明如何在微信云开发控制台中创建所需的数据库集合。

## 创建数据库集合

### 1. 登录云开发控制台

1. 打开微信开发者工具
2. 点击工具栏上的"云开发"按钮
3. 进入云开发控制台

### 2. 创建 person_location 集合

1. 点击左侧"数据库"
2. 点击"+"号创建集合
3. 集合名称：`person_location`
4. 权限设置：选择"仅创建者可读写"或"所有用户可读，仅创建者可写"
5. 点击"确定"

### 3. 创建 locations 集合

1. 再次点击"+"号创建集合
2. 集合名称：`locations`
3. 权限设置：选择"所有用户可读，仅创建者可写"
4. 点击"确定"

### 4. 创建 persons 集合

1. 再次点击"+"号创建集合
2. 集合名称：`persons`
3. 权限设置：选择"所有用户可读，仅创建者可写"
4. 点击"确定"

### 5. 创建索引

为提高查询效率，需要为 `person_location` 集合创建索引：

1. 点击进入 `person_location` 集合
2. 点击"索引管理"
3. 点击"添加索引"
4. 添加以下索引：
   - 索引字段：`location_id`，排序：降序
   - 索引字段：`timestamp`，排序：降序
   - 索引字段：`person_id`，排序：降序

### 6. 创建云存储目录

1. 点击左侧"存储"
2. 点击"创建文件夹"
3. 文件夹名称：`faces`
4. 点击"确定"

## 验证创建结果

创建完成后，应该能看到以下集合：
- person_location
- locations
- persons

云存储应该有：
- faces 文件夹
