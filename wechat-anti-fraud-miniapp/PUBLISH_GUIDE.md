# 微信小程序发布与分享指南

## 发布小程序流程

### 第一步: 完善小程序信息

1. **登录微信公众平台**
   - 访问: https://mp.weixin.qq.com
   - 使用您的微信号登录
   - 进入小程序管理后台

2. **完善基本信息**
   - 小程序名称: 防诈骗线索举报
   - 小程序简介: 提供电信诈骗线索举报服务,共同维护网络安全
   - 小程序头像: 上传正方形logo(建议尺寸: 512*512px)
   - 服务类目: 工具 > 防护工具

3. **设置服务器域名**
   - 进入"开发" → "开发设置"
   - 点击"服务器域名"
   - 无需配置(使用云开发,不需要域名)

### 第二步: 配置小程序参数

1. **修改project.config.json**

确认以下配置正确:
```json
{
  "appid": "wx6b4f24f7e154d8b5",
  "projectname": "anti-fraud-miniapp",
  "libVersion": "2.19.4"
}
```

2. **检查云开发环境**
   - 确认环境ID: `zhuguang-1gctmq3g17561f0c`
   - 确认云函数已部署
   - 确认数据库集合已创建
   - 确认云存储已开通

### 第三步: 上传代码

1. **在微信开发者工具中**
   - 点击工具栏的"上传"按钮
   - 填写版本号: 1.0.0
   - 填写项目备注: 防诈骗线索举报小程序首次发布
   - 点击"上传"

2. **等待上传完成**
   - 上传完成后,会提示"上传成功"
   - 可以在后台看到上传的版本

### 第四步: 提交审核

1. **登录微信公众平台**
   - 进入小程序管理后台
   - 点击"版本管理"

2. **选择开发版本**
   - 找到刚上传的版本
   - 点击"提交审核"

3. **填写审核信息**
   - 功能页面: 首页、举报页、记录页
   - 测试账号: 填写一个可以登录的微信
   - 其他信息: 按提示填写

4. **提交审核**
   - 检查所有信息
   - 点击"提交"
   - 等待审核(通常1-7个工作日)

### 第五步: 发布上线

1. **审核通过后**
   - 登录微信公众平台
   - 点击"版本管理"
   - 找到已审核通过的版本
   - 点击"发布"

2. **发布成功**
   - 小程序正式上线
   - 用户可以通过微信搜索使用

## 分享小程序给别人的方法

### 方法1: 微信搜索(最简单)

1. **用户打开微信**
2. **点击"发现" → "小程序"**
3. **点击右上角的搜索图标**
4. **搜索小程序名称**:
   - 名称: 防诈骗线索举报
   - AppID: wx6b4f24f7e154d8b5

5. **点击搜索结果进入小程序**

### 方法2: 扫描二维码

1. **获取小程序码**
   - 登录微信公众平台
   - 点击"设置" → "基本设置"
   - 点击"小程序码"
   - 下载小程序码图片

2. **分享二维码**
   - 将二维码图片发给朋友
   - 或打印出来贴在公共场所
   - 用户扫描二维码即可使用

### 方法3: 微信分享

**在开发者工具中测试:**
1. 打开小程序
2. 点击右上角"..."
3. 点击"分享给朋友"
4. 选择要分享的好友

**发布后,用户可以这样分享:**
1. 在小程序中
2. 点击右上角"..."
3. 点击"分享给朋友"或"分享到朋友圈"
4. 选择要分享的对象

### 方法4: 群发或公众号推广

1. **公众号文章推广**
   - 在公众号文章中插入小程序卡片
   - 用户点击卡片即可使用

2. **微信群分享**
   - 将小程序分享到微信群
   - 群成员点击即可使用

3. **小程序菜单**
   - 在其他小程序中设置跳转
   - 用户点击菜单进入

### 方法5: 短链接分享

1. **获取短链接**
   - 在微信公众平台生成小程序短链接
   - 格式: https://w.url.cn/s/xxxxx

2. **分享短链接**
   - 通过短信、邮件等方式分享
   - 用户点击链接自动跳转小程序

## 提升小程序曝光率

### 1. 优化小程序名称

**建议:**
- 包含关键词: 防诈骗、举报、线索
- 简短易记: 8-15个字符
- 避免生僻字

### 2. 完善小程序简介

**示例:**
```
专业防诈骗线索举报平台,支持上传图片、视频、聊天记录等证据,
一键举报,及时响应,共同维护网络安全。
```

### 3. 优化小程序头像

**建议:**
- 使用清晰的logo
- 尺寸: 512*512px
- 格式: PNG或JPG
- 大小: 不超过2MB

### 4. 添加关键词标签

在微信公众平台:
- 进入"设置" → "基本设置"
- 添加关键词: 防诈骗、诈骗举报、电信诈骗、反诈

## 常见问题

### Q1: 审核需要多长时间?

A: 通常1-7个工作日,具体看审核情况。

### Q2: 审核不通过怎么办?

A:
1. 查看审核不通过的原因
2. 修改相关问题
3. 重新提交审核

### Q3: 发布后可以修改吗?

A: 可以:
- 功能修改: 需要重新上传和审核
- 文字修改: 部分可以直接修改
- 需要审核: 主要功能修改

### Q4: 如何查看审核进度?

A:
1. 登录微信公众平台
2. 点击"版本管理"
3. 查看审核状态

### Q5: 可以在海外使用吗?

A: 需要额外配置:
- 开通海外云开发环境
- 配置海外服务器域名
- 符合当地法律法规

## 审核注意事项

### 内容要求

1. **必须真实**
   - 不虚假宣传
   - 不夸大效果

2. **必须合法**
   - 不违反法律法规
   - 不传播违法信息

3. **必须有用**
   - 功能完整
   - 用户体验良好

### 技术要求

1. **代码质量**
   - 无明显bug
   - 性能良好
   - 兼容性好

2. **安全要求**
   - 用户信息加密
   - 防止恶意攻击
   - 数据备份

3. **用户体验**
   - 界面美观
   - 操作流畅
   - 加载快速

### 禁止行为

1. **诱导分享**
   - 强制分享才能使用
   - 虚假宣传

2. **收集用户信息**
   - 过度收集
   - 未告知用户

3. **违规内容**
   - 涉黄、涉赌
   - 虚假信息
